# Bookstore App

A modern bookstore/library web app inspired by soft UI design. Built with React.

## Features
- Browse popular books
- Search for books
- Category navigation
- Chat support UI
- Responsive layout

## Project Structure
```
/bookstore-app
  /public
  /src
    /components
      BookCard.jsx
      BookList.jsx
      CategoryBar.jsx
      ChatBox.jsx
      Sidebar.jsx
      SearchBar.jsx
    /assets
    App.jsx
    index.js
  package.json
  README.md
```

## Getting Started
1. Install dependencies: `npm install`
2. Start the app: `npm start`
